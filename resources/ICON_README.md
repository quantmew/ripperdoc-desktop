# Ripperdoc Desktop Icons

This directory contains the application icons for different platforms.

## Required Files

### Windows
- `icon.ico` - Windows application icon (256x256 recommended)

### macOS
- `icon.icns` - macOS application icon bundle

### Linux
- `icons/` - Directory containing PNG icons in various sizes:
  - `16x16.png`
  - `32x32.png`
  - `48x48.png`
  - `64x64.png`
  - `128x128.png`
  - `256x256.png`
  - `512x512.png`

## Creating Icons

1. Start with a high-resolution PNG (1024x1024 recommended)
2. Use tools like:
   - [electron-icon-builder](https://github.com/niccocid/electron-icon-builder)
   - [icon-gen](https://github.com/akabekobeko/npm-icon-gen)
   - Online converters
